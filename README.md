
# Qware Dashboard Project

This project connects to the remote MongoDB and visualizes the `Qware_DashBoard` collection data.

## Prerequisites
- Node.js installed

## Setup
1. Open this folder in VS Code.
2. Open a terminal.
3. Run `npm install` (already done).

## Running the Dashboard
1. Run `node server.js`
2. Open your browser to `http://localhost:3000`

## Structure
- `server.js`: Express server connecting to MongoDB Atlas.
- `public/index.html`: The frontend dashboard using Chart.js.
